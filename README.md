FUSEBOXY INSTALLER (1.x)
========================

## Dependency

Work with composer version `2.2.x`

------------------------------------------------------------------

## Usage

Custom Installer for Composer Installation

`Installer.php` copies the following files to app folder:

*  fuseboxy-core
   *  fusebox_config.php
   *  error_controller.php
   *  home_controller.php
   *  .htaccess
   *  _env.php
   *  index.php
*  fuseboxy-auth
   *  layout.settings.php
*  fuseboxy-layout
   *  layout.settings.php
